# proxy
socket5代理服务器
程序依赖于python3
直接运行不加任何参数即可看到用法
<img width="456" alt="图片" src="https://user-images.githubusercontent.com/121652064/210031145-dc5e57f5-b1c5-4b71-abf1-267d46ca625e.png">
参数"run"，可直接启动代理服务器，端口默认为8888，且为无认证代理
<img width="473" alt="图片" src="https://user-images.githubusercontent.com/121652064/210031213-b46fed75-d9c9-4e0c-a828-0a3e3f4320c6.png">
参数"-p"，可指定代理服务器端口，如端口被占用，或端口需要更改时用到
<img width="476" alt="图片" src="https://user-images.githubusercontent.com/121652064/210031272-6d0b20a6-9443-4d67-8292-af0bf11f61c3.png">
参数"-a"，指定是否需要认证，（注意：开启认证需要浏览器插件，因为浏览器默认设置代理没有输入密码的地方）
